# Content Guide

This file explains how to add more content to movies and series pages.

## Where content is stored now

- `website/assets/data/movies.json` -> movie records
- `website/assets/data/series.json` -> series records
- `docs/` is deployment copy (generated by syncing from `website/`)

## Add a new movie

Open `website/assets/data/movies.json` and add a new object:

```js
{
  title: "Movie Title",
  year: 2024,
  director: "Director Name",
  genre: "Drame Psychologique",
  image: "https://image-url-or-local-path"
}
```

Valid genres currently used:

- `Nouvelle Vague`
- `Drame Psychologique`
- `Polar Français`
- `Comédie Dramatique`

## Add a new series

Open `website/assets/data/series.json` and add a new object:

```js
{
  title: "Series Title",
  genre: "Drame",
  year: 2024,
  episodes: 8,
  image: "https://image-url-or-local-path"
}
```

Valid genres currently used:

- `Drame`
- `Polar`
- `Histoire`
- `Comédie`
- `Téléréalité`

## Pagination behavior

- Movies page: `6` items per page
- Series page: `4` items per page
- Pagination updates automatically after filtering/search.

## ChatGPT Agent Settings

The chatbot in `chatbot.html` uses **OpenAI ChatGPT (gpt-4o-mini)**.

### Configuring the API Key

For the chatbot to work, you must provide an OpenAI API Key.

1. Get your key from the [OpenAI Platform](https://platform.openai.com/api-keys).
2. Open `website/chatbot.html` (and `docs/chatbot.html`).
3. Locate this line in the `<script>` section near the bottom:

   ```js
   const API_KEY = window.OPENAI_API_KEY || "sk-E9YA";
   ```

4. Replace `"sk-E9YA"` with your actual key:

   ```js
   const API_KEY = window.OPENAI_API_KEY || "YOUR_ACTUAL_KEY_HERE";
   ```

> [!WARNING]
> Adding your API key directly to the HTML file makes it visible to anyone who visits your site. For production use, it is recommended to use a backend proxy.

## Publish updated content

1. Update content in `website/`.
2. Sync deployment copy:
   - Windows: `robocopy website docs /E`
3. Commit and push.
