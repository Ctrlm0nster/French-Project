# Content Guide

This file explains how to add more content to movies and series pages.

## Where content is stored now

- `website/assets/data/movies.json` -> movie records
- `website/assets/data/series.json` -> series records
- `docs/` is deployment copy (generated by syncing from `website/`)

## Add a new movie

Open `website/assets/data/movies.json` and add a new object:

```js
{
  title: "Movie Title",
  year: 2024,
  director: "Director Name",
  genre: "Drame Psychologique",
  image: "https://image-url-or-local-path"
}
```

Valid genres currently used:

- `Nouvelle Vague`
- `Drame Psychologique`
- `Polar Français`
- `Comédie Dramatique`

## Add a new series

Open `website/assets/data/series.json` and add a new object:

```js
{
  title: "Series Title",
  genre: "Drame",
  year: 2024,
  episodes: 8,
  image: "https://image-url-or-local-path"
}
```

Valid genres currently used:

- `Drame`
- `Polar`
- `Histoire`
- `Comédie`
- `Téléréalité`

## Pagination behavior

- Movies page: `6` items per page
- Series page: `4` items per page
- Pagination updates automatically after filtering/search.

## Groq AI Settings

The chatbot now uses a secure backend implementation with the **Groq SDK** and **Vercel Serverless Functions**.

### Configuring the API Key

You no longer need to paste your API key directly into the HTML. Instead, follow these steps:

1. **Local Development**:
   - Open your `.env` file.
   - Set the `GROQ_API_KEY` variable:
     ```bash
     GROQ_API_KEY=gsk_your_key_here
     ```
   - The API will now use this key automatically.

2. **Production (Vercel)**:
   - Go to your Vercel project dashboard.
   - Navigate to **Settings** -> **Environment Variables**.
   - Add a new variable:
     - **Key**: `GROQ_API_KEY`
     - **Value**: `gsk_your_actual_key`
   - Vercel will securely provide this key to the `api/chat.js` function.

### How it works

- The frontend (`chatbot.html`) sends requests to `/api/chat`.
- The backend (`api/chat.js`) uses the `groq-sdk` to communicate with Groq using your private key.
- This prevents your API key from being exposed to users visiting your website.

### How to use the AI

- **Ask about movies**: "Quels sont les meilleurs films de la Nouvelle Vague ?"
- **Ask about theaters**: "Où se trouve la Cinémathèque Française ?"
- **General Help**: "Explique-moi l'influence de Jean-Luc Godard."

> [!IMPORTANT]
> Never share your API key or commit it to GitHub. The new system keeps it safe in environment variables.

## Publish updated content

1. Update content in `website/`.
2. Generate deployment copy with environment variables:
   - Run: `npm run build`
3. Commit and push. (Vercel will also run this command automatically).
