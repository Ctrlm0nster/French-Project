# Assets Folder

This folder contains resources for the Cinémathèque website.

## Structure

- `movies/` - Store movie-related files (e.g., descriptions, metadata)
- `series/` - Store series-related files
- `posters/` - Store movie and series poster images
- `data/` - Store data files like JSON lists of movies/series names

## Usage

Add your movie and series names as text files or JSON in the respective folders.
Upload poster images to the `posters/` folder and reference them in the HTML or data files.
