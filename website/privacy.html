Politique de Confidentialité.

Dernière mise à jour : avril 2026. Ce site est une vitrine éditoriale ; les présentes informations visent à être transparentes sur les données traitées.

1. Responsable du traitement

Le site « Tanvi Malik » (projet de démonstration) est édité à titre informatif. Pour toute question relative aux données personnelles, vous pouvez utiliser les coordonnées indiquées dans la section Contact du pied de page.

2. Données collectées

Nous ne vendons pas vos données. Selon votre navigation, des cookies techniques ou des cookies de mesure d’audience peuvent être déposés par l’hébergeur ou des services tiers (par ex. traduction automatique via Google Translate si vous activez ce widget). Les formulaires de recherche sur le site s’exécutent localement ou via des requêtes vers des contenus statiques ; aucun compte utilisateur n’est requis pour consulter le catalogue.

3. Traduction (Google Translate)

Si vous utilisez le sélecteur de langue, le contenu peut être envoyé aux serveurs Google conformément à la politique de confidentialité de Google. Nous n’y avons pas accès.

4. Vos droits

Conformément au RGPD, vous disposez d’un droit d’accès, de rectification, d’effacement, de limitation et d’opposition lorsque le traitement est fondé sur un intérêt légitime. Pour l’exercer, contactez l’éditeur du site.

5. Hébergement

Les fichiers statiques peuvent être hébergés sur une plateforme de type GitHub Pages ou équivalent. Les journaux techniques de l’hébergeur peuvent inclure votre adresse IP et la date d’accès.
